# Dmoney User Management

## Overview

Dmoney is a demo money transaction portal where users can send money, deposit, cashout, and make payments to other users.

## Technology Stack

- **NodeJs** - Runtime environment
- **Postman** - API testing
- **Newman** - Command-line collection runner
- **HTML Report Extra** - Test reporting

## Getting Started

### Prerequisites

⚠️ **Important:** You must have the secret key to run this project.

### Installation & Testing

```bash
# Install dependencies
npm i

# Run tests
npm test
```
