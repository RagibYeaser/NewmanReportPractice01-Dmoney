# Project Title: Dmoney user management
## Project Description: Dmoney is a demo money transaction portal where users can send money, deposit, cashout and payment to other users.

### Technology used:
- NodeJs
- Postman
- Newman
- HTML Report Extra

### How to run:
- `npm i`
- `node report.js`
